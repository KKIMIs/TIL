# 브루트포스 

## 브루트 포스란?

: 검색 대상이 되는 원본 문자열의 처음부터 끝까지 차례대로 순회하며 문자들을 일일이 비교하는 방식의 알고리즘

## 비교과정
 - 원본 문자열 T, 찾고자 하는 문자열 P
 - 첫번째 문자부터 하나씩 비교
   - 같으면 인덱스 이동
   - 다르면 p는 맨 처음 인덱스로 이동 
 - 반복
